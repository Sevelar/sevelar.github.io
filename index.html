Introduction

Projects

Discord Previewer is a live-preview markdown text editor, made in Node.js and Electron. It supports features like custom avatars, custom nicknames and role colors. It was my first attempt at making such application.
Eventually I abandoned it in favor of DisViewer.

DisViewer is a reworked version of Discord Previewer. Instead of using the pure JavaScript, I chose React as UI framework for such project. Made from scratch, featuring completely refreshed CSS and improved performance
Still in development.

Astron is my main goal. A full-blown chat application with various features. Instant messaging, voice calling, customizations and a lot more. I wish that one day Astron could become a big service, giving some freshness on a market. However it’s a far goal, so the initial release of Astron is intended to be self-hosted. As for now, I’m working on a proof-of-concept demo, which is going to be public.
